Help & Guide

Getting Started

  1. Click "Start AI" to begin object detection
  2. Allow camera access when prompted
  3. Watch the live detection in real-time
  4. Click "Listen" to ask AI questions
  5. Check Analytics for usage statistics

Voice Commands

  • "What do you see?" - List all detected objects
  • "How many people?" - Count specific objects
  • "Is it safe?" - Check for potential hazards
  • "Where is the nearest object?" - Get proximity info

Features

  • Real-time Detection: 100+ object types including tech items
  • Technology Recognition: Laptops, projectors, smartboards
  • Distance Estimation: Proximity calculations in meters
  • Voice Q&A: Intelligent response system
  • Analytics: Usage tracking and statistics

Troubleshooting

  • Camera not starting: Check permission and ensure no other app uses it
  • Model not loading: Check internet connection, disable VPN/ad-block
  • No detections: Increase sensitivity in Settings or improve lighting
  • Audio not speaking: Enable Voice Feedback and raise Volume

Keyboard Shortcuts

  • S - Start AI
  • X - Stop AI
  • L - Toggle Listen

Privacy & Permissions

  • Video stays on your device; processing runs locally in the browser
  • Analytics are session-only; no data is stored or uploaded
  • Microphone access is used only when you click Listen
